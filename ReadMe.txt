Keyboard shortcuts
    DEL                 Delete active node
    F2                  Edit active node
    F3                  Connect to server
    ENTER               Connect to server
    DBL+Click           Connect to server    
    INSERT              Insert sub host
    +INSERT             Insert host
    ^%+Left/Right       Focus tab, for switching
    type any string     Incremental search  

Legend
    %: Alt
    ^: Ctrl
    +: Shift
